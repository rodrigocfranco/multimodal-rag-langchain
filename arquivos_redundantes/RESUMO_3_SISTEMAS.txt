    1️⃣  SISTEMA SEPARADO (1 PDF = 1 Vectorstore)
═══════════════════════════════════════════════════════════════════════════

COMO FUNCIONA:
  PDF 1 → vectorstore_1/
  PDF 2 → vectorstore_2/
  PDF 3 → vectorstore_3/
  
  Pergunta → Escolhe qual PDF consultar

SCRIPTS:
  • processar_e_salvar.py
  • consultar_vectorstore.py

QUANDO USAR:
  ✅ PDFs muito diferentes (medicina vs programação)
  ✅ Quer isolar conhecimentos
  ✅ Teste individual de PDFs

EXEMPLO:
  python processar_e_salvar.py "diabetes.pdf"
  python consultar_vectorstore.py diabetes
  🤔 Pergunta sobre diabetes

COMO FUNCIONA:

  • consultar_com_filtros.py

QUANDO USAR:
  ✅ PDF grande (>50 páginas)
  ✅ Precisa filtrar (só tabelas, só página X)
  ✅ Análises avançadas
  ✅ Debugging detalhado
  ✅ Uso acadêmico/profissional

EXEMPLO:
  python processar_com_metadata_avancado.py "tese.pdf"
  python consultar_com_filtros.py tese_metadata
